## 简介

使用网易云音乐API + 波点音乐UI实现的音乐项目。

后端是这位大佬的: https://github.com/Binaryify/NeteaseCloudMusicApi

## 特点
* 使用MVVM架构模式
* Retrofit2 + 协程完成的网络请求
* 内置换肤框架
* 使用了MotionLayout
